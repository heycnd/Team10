# 需求分析
大廈管理系統的需求分析：
> 1.管理者可利用後台做公告的發布、意見的管理、設備的管理及人員的管控。
> 
> 2.使用者可透過網頁觀看公告、提出意見以及帳單的查詢。
> 
> 3.後台可即時通報管理員，使用者的報修情形。
> 
> 4.後台可即時的反應保全及清潔人員的工作進度。
